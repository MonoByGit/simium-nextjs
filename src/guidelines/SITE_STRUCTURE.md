# Simium Website - 16 Core Pages Structure

## Core Website Architecture (16 Pages)

### 1. **Landing & Discovery (4 pages)**
- **HomePage** (`/`) - Main landing page
- **ProductsPage** (`/producten`) - All scans overview  
- **AboutPage** (`/over-simium`) - About Simium
- **ContactPage** (`/contact`) - Contact information

### 2. **Free Scans (3 pages)**
- **CashflowanalysePage** (`/cashflow-analyse`) - Cashflow scan
- **CloudkostenscanPage** (`/cloudkostenscan`) - Cloud costs scan  
- **PrijsstrategieCheckPage** (`/prijsstrategie-check`) - Pricing strategy scan

### 3. **Premium Analysis (3 pages)**
- **CashflowVerdiepingPage** (`/cashflow-verdieping`) - Premium cashflow analysis
- **CloudkostenVerdiepingPage** (`/cloudkosten-verdieping`) - Premium cloud analysis
- **PrijsstrategieVerdiepingPage** (`/prijsstrategie-verdieping`) - Premium pricing analysis

### 4. **Results Dashboards (3 pages)**
- **CashflowResultatenPage** (`/cashflow-resultaten`) - Cashflow results dashboard
- **CloudkostenResultatenPage** (`/cloudkosten-resultaten`) - Cloud results dashboard  
- **PrijsstrategieResultatenPage** (`/prijsstrategie-resultaten`) - Pricing results dashboard

### 5. **Social Proof & Auth (3 pages)**
- **VoorbeeldrapportPage** (`/voorbeeldrapport`) - Example report
- **TestimonialsPage** (`/testimonials`) - Customer stories
- **LoginPage** (`/inloggen`) - Login & Registration (combined)

## Utility Pages (Not counted in core 16)
- **PrivacyPage** (`/privacy`) - Privacy policy
- **TermsPage** (`/voorwaarden`) - Terms & conditions
